Para Ejecutar el sistema es necesario las siguientes librerías adicionales: (npm install)
    "angularfire2": "^5.2.1",
    "cordova-android": "8.0.0",
    "cordova-plugin-datepicker": "^0.9.3",    
    
Para instalar esta libreria fue necesario forzar la instalación (npm install --force)
    "firebase": "^6.3.4"

Es necesario settear las credenciales de firebase en el archivo "environment.ts"
